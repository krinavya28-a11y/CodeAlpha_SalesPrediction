# Sales Prediction using Python

## Project Overview

This project predicts product sales based on advertising expenditure across different marketing platforms.

The objective is to understand how advertising budgets influence sales and to build a machine learning model capable of forecasting future sales.

## Dataset Features

- TV Advertising Budget
- Radio Advertising Budget
- Newspaper Advertising Budget

### Target Variable

- Sales

## Technologies Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn
- Google Colab

## Machine Learning Algorithm

- Linear Regression

## Project Workflow

1. Data Collection
2. Data Cleaning
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Sales Prediction
7. Model Evaluation
8. Data Visualization

## Model Performance

- R² Score: 0.8994

The model successfully predicts sales using advertising expenditure data and demonstrates strong predictive performance.

## Key Insights

- TV advertising has a significant impact on sales.
- Radio advertising also contributes positively to sales growth.
- Advertising spending can be used to estimate future sales outcomes.
